      • Method Detail

        • void checkActiveBundles(String groupProperty, SlingBundleSupport slingBundleSupport)

          If any of the bundles on the server pointed to by "slingBundleSupport" are inactive and their symbolic name contains "groupProperty", then throws a org.gradle.api.GradleException; otherwise simply returns.

          It will poll the server every retryWaitMs up to # # getMaxWaitMs ( ) maxWaitMs to see if the state has changed to be ACTIVE before throwing an exception.

          throws:
          GradleException after polling, the bundles still are not ACTIVE
          Parameters:
          groupProperty - the part of the symbolic name to look for (e.g., "com.myco")
          slingBundleSupport - the server and connection to check
